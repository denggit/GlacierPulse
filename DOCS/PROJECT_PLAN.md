# GlacierPulse 项目计划

## 项目概述
GlacierPulse 是一个针对 OKX 交易所 ETH/USDT 永续合约的量化交易系统。系统通过三层策略模块，识别机构建仓信号、捕捉市场转折点、执行高概率突破交易。

## 核心策略阶段

### 第一阶段：机构建仓检测
- **冰山订单探测 (Iceberg Order Detection)**：识别大单拆散后的隐蔽建仓行为
- **扫损流动性清扫 (Stop-Hunt Liquidity Sweep)**：探测机构故意触发止损单以获取廉价筹码的行为

### 第二阶段：市场转折点捕捉
- **波动率压缩 (Volatility Compression)**：监测多空双方能量耗尽后的微观价差缩小
- **流动性真空 (Liquidity Vacuum)**：识别盘口进入"死寂"状态，等待新方向的数学特征

### 第三阶段：交易执行引擎
- **定向失衡点火 (Directional Imbalance Ignition)**：捕捉盘口失衡引发的方向性突破
- **动量突破 (Momentum Breakout)**：确认趋势形成后的开仓信号

## 未来扩展
- **Volume Profile**：添加交易量分布分析，用于精确定位交易位置和止盈止损位

## 备注
- 具体实现逻辑将在后续开发阶段详细讨论
- 系统架构基于现有代码基础（数据流、执行器、通知模块）进行扩展
- 采用异步Python架构，保持高性能实时处理能力
